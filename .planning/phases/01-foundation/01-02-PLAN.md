    2. **Migration with RLS + pgcrypto functions** — After running `prisma migrate dev --name init`, append to the generated SQL:
       ```sql
       -- Enable pgcrypto
       CREATE EXTENSION IF NOT EXISTS pgcrypto;

       -- PII encryption helper functions (LGPD-04)
       -- These wrap pgp_sym_encrypt/decrypt so application code uses consistent key from env
       CREATE OR REPLACE FUNCTION encrypt_pii(plaintext TEXT, key TEXT)
       RETURNS TEXT AS $$
         SELECT encode(pgp_sym_encrypt(plaintext, key), 'base64');
       $$ LANGUAGE SQL IMMUTABLE;

       CREATE OR REPLACE FUNCTION decrypt_pii(ciphertext TEXT, key TEXT)
       RETURNS TEXT AS $$
         SELECT pgp_sym_decrypt(decode(ciphertext, 'base64'), key);
       $$ LANGUAGE SQL IMMUTABLE;

       -- Enable RLS on all tenant-scoped tables
       ALTER TABLE "User" ENABLE ROW LEVEL SECURITY;
       ALTER TABLE "ConsentRecord" ENABLE ROW LEVEL SECURITY;
       ALTER TABLE "ErasureRequest" ENABLE ROW LEVEL SECURITY;

       -- RLS policies (CRITICAL: TRUE as second arg to current_setting)
       CREATE POLICY tenant_isolation ON "User"
         USING ("clinicId" = current_setting('app.current_tenant', TRUE)::uuid);
       CREATE POLICY tenant_isolation ON "ConsentRecord"
         USING ("clinicId" = current_setting('app.current_tenant', TRUE)::uuid);
       CREATE POLICY tenant_isolation ON "ErasureRequest"
         USING ("clinicId" = current_setting('app.current_tenant', TRUE)::uuid);
       ```

       NOTE: Clinic table does NOT have RLS — it's the root tenant table.

    3. **PII crypto helpers** (`packages/database/src/crypto.ts`):
       - Export `encryptPii(prisma: PrismaClient, plaintext: string): Promise<string>` — calls `SELECT encrypt_pii($1, $2)` via `prisma.$queryRawUnsafe` using `process.env.PGCRYPTO_KEY`
       - Export `decryptPii(prisma: PrismaClient, ciphertext: string): Promise<string>` — calls `SELECT decrypt_pii($1, $2)` via `prisma.$queryRawUnsafe` using `process.env.PGCRYPTO_KEY`
       - Throws if PGCRYPTO_KEY env var is not set
